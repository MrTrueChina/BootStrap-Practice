input type="radio"：单选按钮

radio类：bootstrap的单选按钮的美化类，使用时加在div上，之后在div里面加一个label，最后写单选按钮

radio类会自动对齐文字和按钮，但这个功能只有在 label 的包裹中才有效，否则是完全不同的处理方式

name=""：设定名字，同样名字的单选按钮视为一组，选择一个其他的自动取消


input type="checkbox"：复选框

checkbox类：bootstrap提供的复选框的美化类，使用时也是写在div里，套一个label，最后写复选框

name=""：设定名字，同样名字的复选框视为一组，不过这次不会选一个自动取消别的


radio-inline类：bootstrap提供的用来定义内联单选按钮的类，这个类修饰的radio标签不会换行

使用这个类的时候直接写在label上，然后里面写radio就可以正常生效


checkbox-inline类：botstrap提供的用来定义内联复选框的类，这个类修饰的checkbox不会换行

使用这个类时同样直接写在label上
