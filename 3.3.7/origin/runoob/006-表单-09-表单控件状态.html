表单控件的状态

默认状态下输入框是可用的，点击后获得焦点，显示发光效果（然而这个效果在css里叫“shadow”，影子）


添加 disabled="disabled" 属性可以禁用控件


通过用 <fieldset disabled="disabled"> 标签包围表单控件，可以一次禁用多个控件


不仅是 input 标签，其他表单控件都是可以禁用的

这个 select 和前面的文本框是通过同一个 fieldset 禁用的


使用 has-success 类可以定义“成功”的输入控件，成功可以是指符合标准或其他的一些事情，当然随便用也可以

视觉效果是文字变绿色、输入控件整体颜色都变成绿色


使用 has-warning 类可以定义“警告”的输入控件，像成功一样，可以表示有意义的信息，也可以随便用，甚至用作装饰

视觉效果是文字和控件的整体颜色变成黄色


使用 has-error 类可以定义“错误”的输入控件，可以表示有意义的信息，也可以随便用，甚至用作装饰

视觉效果是文字和控件的整体颜色变成红色
