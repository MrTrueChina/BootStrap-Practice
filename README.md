### 千变万变，前端不变，人生苦短，我学前端
Java一时爽，配置火葬场；Boot一时爽，转岗火葬场。在我们组的项目（[V-ray-Shop](https://github.com/MrTrueChina/V-ray-Shop)）因为没有前端黄了之后，我们深刻的意识到了残酷的现实：看脸的世界没有前端是不行的，看脸的世界没有前端是万万不能的。为了在看脸的世界存活，我要学好看的前端。
